### 🌳 Tree Component

This repository contains an Angular implementation of a dynamic tree structure with a built-in search algorithm. The tree supports nested nodes and allows users to search for a value, automatically expanding and highlighting matching nodes.


#### 🚀 Run the application

```bash
git clone https://github.com/Elie-Soued/TreeComponent.git
cd ogs
npm install
ng serve

```

Open your browser at http://localhost:4200/ to see the app.  


##### 🧪 Test the application 

Type a value into the search bar.

If a match is found:

- The tree will expand all ancestor nodes leading to the match. The matched node(s) will be highlighted in green.

- If no match is found, the tree remains unchanged.


##### 🛠 Technologies Used

Angular

Angular Material – Tree module
