### Tree Component

This repo is the implementation of a tree structure and a search algorithm that traverse the tree to look for the searched value. 



#### Run the application

```bash
git clone https://github.com/Elie-Soued/TreeComponent.git
cd ogs
ng serve

```

Open your browser at http://localhost:4200/ to see the app.  


##### Test the application 

Enter a value in the search bar.
If the value is present in the tree, the tree node ancestor will expand and the value will be highlighted in green.
